ETL Pipeline & Reporting

Context: BIAr Consulting Role: Data Engineer

Problem

Business teams required consistent, time-based reports to track performance, but existing data flows were fragmented and difficult to maintain.

What I did

  • Designed structured ETL pipelines using Informatica PowerCenter
  • Built custom SQL views for reporting and analytics
  • Standardized execution periods to enable historical comparison

Outcome

Improved data reliability and reduced manual reporting effort, enabling stakeholders to make faster, data-driven decisions.

Database Migration: PL/SQL → PostgreSQL

Context: BIAr Consulting Focus: Data integrity & performance

Problem

Legacy Oracle-based systems needed to be migrated to PostgreSQL without breaking existing reporting workflows.

What I did

  • Translated complex PL/SQL logic into PostgreSQL-compatible queries
  • Optimized queries to maintain performance parity
  • Validated results to ensure data consistency across systems

Outcome

Successful migration with minimal disruption, supporting the organization’s transition to a modern data stack.
